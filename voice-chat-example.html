Voice Chat Component Examples

This page demonstrates different ways to use the Voice Chat Component.

Example 1: Basic Usage (Bottom Right)

A simple floating microphone button in the bottom right corner.

<script src="voice-chat-component.js"></script> <script> const voiceChat1 = new VoiceChatComponent({ wsUrl: 'wss://your-backend-url.com/realtime', containerId: 'voice-chat-1', position: 'bottom-right', size: 'medium' }); </script>

Example 2: Bottom Left, Large Size

A larger microphone button positioned in the bottom left.

<script> const voiceChat2 = new VoiceChatComponent({ wsUrl: 'wss://your-backend-url.com/realtime', containerId: 'voice-chat-2', position: 'bottom-left', size: 'large' }); </script>

Example 3: Inline Usage

Component embedded directly in page content.

<div id="inline-voice-chat"></div> <script> const voiceChat3 = new VoiceChatComponent({ wsUrl: 'wss://your-backend-url.com/realtime', containerId: 'inline-voice-chat', showModal: false, size: 'large' }); </script>

Example 4: No Modal, Direct Chat

Direct voice chat without modal popup.

<script> const voiceChat4 = new VoiceChatComponent({ wsUrl: 'wss://your-backend-url.com/realtime', containerId: 'voice-chat-4', position: 'top-right', size: 'small', showModal: false }); </script>

Configuration Options

  • wsUrl: WebSocket URL for your backend
  • containerId: ID for the container element
  • position: 'bottom-right', 'bottom-left', 'top-right', 'top-left'
  • size: 'small', 'medium', 'large'
  • showModal: true/false - show modal on click
  • debug: true/false - enable debug logging

Public Methods

  • connect(): Manually connect to WebSocket
  • disconnect(): Disconnect and cleanup
  • setWebSocketUrl(url): Update WebSocket URL
  • destroy(): Completely remove component
